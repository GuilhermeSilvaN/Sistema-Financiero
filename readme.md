Sistema web desenvolvido para auxiliar no gerenciamento e organização das finanças pessoais.

O projeto reúne uma aplicação frontend, responsável pela interface utilizada pelo usuário, e uma API backend, responsável pelo processamento e gerenciamento dos dados financeiros.

## Funcionalidades

O sistema tem como objetivo permitir o gerenciamento de informações financeiras, como:

* Controle de entradas;
* Registro de despesas;
* Visualização do saldo;
* Organização das informações financeiras;
* Acompanhamento dos dados por meio de um dashboard.

## Tecnologias utilizadas

* Java;
* Spring Boot;
* JavaScript;
* React;
* CSS;
* HTML.

## Estrutura do projeto

```text
Sistema-Financiero/
│
├── api-financeiro/
│   └── Backend da aplicação
│
└── front-financeiro/
    └── Interface da aplicação
```

O backend é responsável pelo processamento das informações e comunicação com o banco de dados. O frontend fornece a interface utilizada pelo usuário para visualizar e gerenciar seus dados financeiros.

## Funcionamento

```text
Usuário
   ↓
Interface do sistema
   ↓
API
   ↓
Banco de Dados
```

A aplicação permite que o usuário interaja com suas informações financeiras por meio da interface web, enquanto a API processa e armazena os dados.

## Objetivo

O projeto foi desenvolvido para praticar conceitos de desenvolvimento full stack, integração entre frontend e backend, criação de APIs e gerenciamento de dados.
